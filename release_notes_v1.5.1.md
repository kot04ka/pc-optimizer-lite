# PC Optimizer Lite 1.5.1

Focus: fit-and-finish fixes for the redesigned UI and stronger Windows visual-effects control.

Changes:
- Fixed settings layout overflow: the settings page no longer creates a horizontal scrollbar at 1215px-wide windows.
- Limited settings input/combo widths and shortened long checkbox labels while keeping details in tooltips.
- Replaced the tiny `...` topbar menu with a clear "Действия" button and a normal drop-down menu.
- Expanded low-power Windows visual effects handling to include UI effects, fade effects, shadows, hot tracking, gradient captions, smooth scrolling, and animations.
- Visual effects are restored when the option is disabled or when the app exits.

Safety notes:
- The app does not disable Windows Defender, antivirus, or system protection settings.
- Visual-effects changes are reversible and scoped to Windows appearance/performance flags.
